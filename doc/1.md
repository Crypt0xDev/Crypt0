# Estructura de Proyecto Astro con Soporte Multilenguaje y Modo Oscuro
```
├── public/
│   ├── favicon.ico
│   └── theme.js                         # Gestión modo oscuro / claro (detectar y toggle)
├── src/
│   ├── assets/
│   │   ├── css/
│   │   │   ├── global.css               # Tailwind base + reset + diseño general
│   │   │   └── components/              # CSS específicos (header, buttons, cards)
│   ├── components/
│   │   ├── ui/                         # Componentes UI reutilizables
│   │   │   ├── Button.astro            # Botón estilizado con icono
│   │   │   ├── IconCard.astro          # Cards de "Next steps"
│   │   │   ├── Header.astro            # Navegación + selector idioma + tema
│   │   │   ├── LanguageSelector.astro  # Toggle de idiomas
│   │   │   └── ThemeToggle.astro       # Botón toggle modo oscuro
│   │   ├── layouts/
│   │   │   ├── Layout.astro            # Layout base (SEO, html, head, body)
│   │   │   ├── PageLayout.astro        # Layout para páginas estáticas o landing
│   │   │   └── (opcional) BlogLayout/WriteupLayout si crece contenido
│   │   └── widgets/                    # Pequeños componentes auxiliares (buscador, footer, etc)
│   ├── content/
│   │   ├── i18n/                      # Diccionarios JSON (inglés por defecto)
│   │   │   ├── en.json
│   │   │   └── es.json
│   │   └── posts/                      # Contenido (markdown, json, mdx)
│   │       ├── en/                    # Posts y writeups en inglés
│   │       └── es/                    # Posts y writeups en español
│   ├── pages/
│   │   ├── index.astro                 # Redirección a /en/ (idioma por defecto)
│   │   ├── [lang]/                    # Páginas localizadas (lang = en, es)
│   │   │   ├── index.astro             # Home landing page (multilenguaje)
│   │   │   ├── about.astro             # About page para cada idioma
│   │   │   └── writeups/
│   │   │       ├── index.astro         # Lista de writeups
│   │   │       └── [slug].astro        # Writeup individual (opcional)
│   ├── scripts/                        # Scripts vite + helpers (por ejemplo i18n, tema)
│   │   ├── i18n.ts                    # Funciones para cargar y traducir textos
│   │   └── utils.ts                   # Utilidades varias (fechas, slug, etc)
│   └── styles/
│       ├── tailwind.config.cjs        # Configuración de Tailwind
│       ├── reset.css                  # Normalize.css personalizado
│       └── variables.css              # Variables CSS / tokens de diseño
├── astro.config.mjs                  # Configuración Astro general
├── package.json                     # Dependencias y scripts
├── tsconfig.json                    # TypeScript configuración
└── README.md                        # Documentación básica
```

# Estructura Simplificada de Proyecto Astro con Multilenguaje y Modo Oscuro
```
├── public/
│   ├── theme.js                      # Script para modo oscuro automático y toggle manual
│   └── assets/                       # Imágenes, iconos y recursos estáticos
├── src/
│   ├── assets/
│   │   └── styles/
│   │       ├── global.css            # Tailwind base + reset + variables
│   │       └── components/           # CSS específicos por componente (Botones, Cards)
│   ├── components/
│   │   ├── ui/
│   │   │   ├── Button.astro          # Botón estilizado
│   │   │   ├── IconCard.astro        # Cards para cada "Next step"
│   │   │   ├── Header.astro          # Header con navegación y selectores (idioma, tema)
│   │   │   └── LanguageSwitcher.astro # Selector idioma con persistencia
│   │   ├── layouts/
│   │   │   ├── Layout.astro          # Layout base: html, head, body, carga theme.js
│   │   │   └── PageLayout.astro      # Layout para páginas (usa Header y slot)
│   │   └── icons/
│   │       └── icon-sprite.svg       # Definición y símbolos svg de iconos
│   ├── content/
│   │   ├── i18n/
│   │   │   ├── en.json               # Diccionario UI en inglés (default)
│   │   │   └── es.json               # Diccionario UI en español
│   └── pages/
│       ├── index.astro               # Redirecciona a /en/ (idioma por defecto)
│       ├── [lang]/
│       │   └── index.astro           # Página principal multilenguaje
│       └── api/                      # API routes si las necesitas
├── tailwind.config.cjs               # Configuración Tailwind CSS con darkMode: 'class'
├── astro.config.mjs                  # Config Astro general
├── package.json
└── README.md                       # Documentación básica
```
