# Imágenes del Blog

Coloca aquí las imágenes para tus posts de blog.

## Uso

En el frontmatter de tu post markdown, añade:

```markdown
---
title: 'Mi Post'
heroImage: '/images/blog/mi-imagen.jpg'
---
```

## Formato Recomendado

- **Formato:** JPG, PNG o WebP
- **Dimensiones:** 1200x630px (ratio 1.91:1 para OG image)
- **Tamaño:** < 200KB optimizado
- **Nomenclatura:** kebab-case (ejemplo: `intro-ciberseguridad.jpg`)

## Optimización

Puedes usar herramientas como:
- [TinyPNG](https://tinypng.com/) - Compresión de imágenes
- [Squoosh](https://squoosh.app/) - Optimizador de Google
- [ImageOptim](https://imageoptim.com/) - Para macOS

## Estructura

```
public/images/blog/
├── intro-ciberseguridad.jpg
├── tutorial-nmap.png
└── ...
```
